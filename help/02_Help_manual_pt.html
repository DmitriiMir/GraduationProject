Sobre o Programa
Este aplicativo foi desenvolvido para exibir vários dados geofísicos. Ele permite criar gráficos, histogramas e superfícies 3D de alta qualidade. Qualquer dado numérico em formato tabular salvo em um arquivo CSV pode ser usado como entrada.
Interface do Programa
  • Clique no botão "Abrir" para encontrar e abrir o arquivo CSV de origem.
  • Use o menu suspenso para selecionar os dados relevantes.
  • Para limpar os dados, clique no botão "Limpar".
  • Para realizar plotagens gráficas, clique no botão "Mostrar".
  • Para mudar o idioma da interface, selecione um dos idiomas disponíveis no menu suspenso "Seleção de Idioma".
  • Se tiver alguma dúvida, pode procurar ajuda clicando no botão "Ajuda".
Gráficos
Gráficos simples são criados usando a biblioteca científica Matplotlib. Os seguintes tipos de gráficos estão implementados neste programa:
  • Gráfico simples. Para plotar um gráfico simples, selecione os valores de uma das duas colunas: "Leituras do Instrumento 1" ou "Leituras do Instrumento 2".
  • Gráfico combinado. Para comparar as leituras de dois instrumentos, selecione os valores de duas colunas: "Leituras do Instrumento 1" e "Leituras do Instrumento 2".
  • Gráfico dependente da distância. Você também pode combinar várias opções de gráfico dependendo da distância. Para isso, selecione o valor do canal de distância no menu suspenso "Distância".
Histogramas
A biblioteca Seaborn é usada para criar histogramas de distribuição normal. Várias opções para exibir a distribuição normal estão implementadas neste programa:
  • Histograma simples sobreposto com uma curva de distribuição normal. Para plotar, selecione os valores de uma das duas colunas: "Leituras do Instrumento 1" ou "Leituras do Instrumento 2".
  • Histogramas combinados. Para comparar leituras, selecione valores para duas colunas: "Leituras do Instrumento 1" e "Leituras do Instrumento 2".
  • Distribuição de valores dependendo da distância. Você também pode combinar várias opções de exibição dependendo da distância. Para isso, selecione o valor do canal de distância no menu suspenso "Distância".
Trabalhando com a Janela Matplotlib
A janela de gráficos científicos Matplotlib permite ao usuário realizar várias ações, como:
  • Arrastar o gráfico;
  • Escalar;
  • Exibir a área de plotagem;
  • Alterar rótulos dos eixos;
  • Alterar cores;
  • Salvar o gráfico em formato selecionado, etc.
Superfície 3D
A biblioteca Plotly é usada para criar superfícies 3D. Para criar uma superfície, selecione todas as 3 colunas: X, Y e Z. !!!Todas as coordenadas devem estar no sistema retangular!!!
Trabalhando com a Janela de Superfície 3D
A janela de exibição da Superfície 3D permite ao usuário realizar várias ações, como:
  • Arrastar;
  • Escalar;
  • Girar;
  • Ler dados;
  • Salvar a imagem no formato selecionado, etc.
Integração com OasisMontaj
Este aplicativo permite trabalhar com qualquer arquivo CSV. No entanto, como este programa foi encomendado pela AO "Yuzhmorgeologia", vamos discutir o processo de exportação de um banco de dados do OasisMontaj para um arquivo CSV. Os passos no ambiente OasisMontaj são os seguintes:
  • Banco de dados / Exportar / CSV etc...;
  • Selecione os parâmetros de salvamento;
  • Clique em "Ok".
