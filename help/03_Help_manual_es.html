Acerca del Programa
Este programa fue desarrollado para visualizar varios datos geofísicos. Permite crear gráficos, histogramas y superficies 3D de alta calidad. Cualquier dato numérico en formato tabular guardado en un archivo CSV puede ser utilizado como entrada.
Interfaz del Programa
  • Haz clic en el botón "Abrir" para encontrar y abrir el archivo CSV de origen.
  • Utiliza el menú desplegable para seleccionar los datos relevantes.
  • Para limpiar los datos, haz clic en el botón "Limpiar".
  • Para realizar gráficos, haz clic en el botón "Mostrar".
  • Para cambiar el idioma de la interfaz, selecciona uno de los idiomas disponibles en el menú desplegable "Selección de Idioma".
  • Si tienes alguna pregunta, puedes buscar ayuda haciendo clic en el botón "Ayuda".
Gráficos
Los gráficos simples se crean utilizando la biblioteca científica Matplotlib. Los siguientes tipos de gráficos están implementados en este programa:
  • Gráfico simple. Para trazar un gráfico simple, selecciona los valores de una de las dos columnas: "Lecturas del Instrumento 1" o "Lecturas del Instrumento 2".
  • Gráfico combinado. Para comparar las lecturas de dos instrumentos, selecciona los valores de dos columnas: "Lecturas del Instrumento 1" y "Lecturas del Instrumento 2".
  • Gráfico dependiente de la distancia. También puedes combinar varias opciones de gráficos dependiendo de la distancia. Para ello, selecciona el valor del canal de distancia en el menú desplegable "Distancia".
Histogramas
La biblioteca Seaborn se utiliza para crear histogramas de distribución normal. Se implementan varias opciones para mostrar la distribución normal en este programa:
  • Histograma simple superpuesto con una curva de distribución normal. Para trazar, selecciona los valores de una de las dos columnas: "Lecturas del Instrumento 1" o "Lecturas del Instrumento 2".
  • Histogramas combinados. Para comparar lecturas, selecciona valores para dos columnas: "Lecturas del Instrumento 1" y "Lecturas del Instrumento 2".
  • Distribución de valores dependiendo de la distancia. También puedes combinar varias opciones de visualización dependiendo de la distancia. Para ello, selecciona el valor del canal de distancia en el menú desplegable "Distancia".
Trabajo con la Ventana Matplotlib
La ventana de gráficos científicos Matplotlib permite al usuario realizar varias acciones, como:
  • Arrastrar el gráfico;
  • Escalar;
  • Mostrar el área de trazado;
  • Cambiar etiquetas de ejes;
  • Cambiar colores;
  • Guardar el gráfico en el formato seleccionado, etc.
Superficie 3D
La biblioteca Plotly se utiliza para crear superficies 3D. Para crear una superficie, selecciona las 3 columnas: X, Y y Z. !!!Todas las coordenadas deben estar en el sistema rectangular!!!
Trabajo con la Ventana de Superficie 3D
La ventana de visualización de la Superficie 3D permite al usuario realizar varias acciones, como:
  • Arrastrar;
  • Escalar;
  • Girar;
  • Leer datos;
  • Guardar la imagen en el formato seleccionado, etc.
Integración con OasisMontaj
Esta aplicación permite trabajar con cualquier archivo CSV. Sin embargo, dado que este programa fue encargado por AO "Yuzhmorgeologia", vamos a discutir el proceso de exportación de una base de datos de OasisMontaj a un archivo CSV. Los pasos en el entorno de OasisMontaj son los siguientes:
  • Base de datos / Exportar / CSV, etc...;
  • Selecciona los parámetros de guardado;
  • Haz clic en "Ok".
